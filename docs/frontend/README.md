# Frontend Topic Map

Use this directory when the top-level frontend index is still too broad.

## Files

- `app-shell-and-routing.md`: stack, build/runtime commands, layout shell, and route map.
- `client-and-state.md`: API client, shared types, query keys, and invalidation rules.
- `workspaces.md`: entries, groups, accounts, properties, dashboard, and settings pages.
- `agent-workspace.md`: thread panel, timeline, review modal, and composer behavior.
- `styles-and-operations.md`: shared components, feature-module map, styling system, operational impact, and constraints.

## Fastest Path By Question

- "How is the app shell composed?" -> `app-shell-and-routing.md`
- "Where should query or API code live?" -> `client-and-state.md`
- "How does a page or workspace behave?" -> `workspaces.md`
- "How does the AI page work?" -> `agent-workspace.md`
- "What shared components or CSS patterns already exist?" -> `styles-and-operations.md`
