# Completed Task Archive

This directory stores archived implementation task docs after the work is finished.

- Use dated filenames in the form `YYYY_MM_DD-task_name.md`.
- Keep durable behavior and architecture details in the canonical docs under `docs/`, `backend/docs/`, `frontend/docs/`, and `docs/features/`.
- Treat files here as historical context, not source-of-truth for current behavior.
