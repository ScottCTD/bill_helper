Currently we fill the form to create an entry, and click the name of any entry and enter the edit page to edit an entry.

A more convenient way to do this is to have a dedicated edit page (like when you click a notion page of a notion db, the page will popup) popup that's responsible for both creating and editing entries.

Similar to how Notion page works, the popup page should allow editing properties and have a Markdown editor.

You should also remove the "attachments" feature and make the Markdown editor support image. If I copy an image in, the Markdown editor should display the image. You may use a library for this. 
I recommend this Notion-like editor library: https://github.com/TypeCellOS/BlockNote
