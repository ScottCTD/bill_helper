# iOS Docs

This directory holds iOS-client-specific documentation that should stay close to the native app code under `ios/`.

Start with `../README.md` for the current shipped iOS MVP summary and `ios/` folder map, then use the files here for screen- and flow-level detail.

## Files

- `agent_mvp.md`: agent thread list/detail, upload, run-state, and review behavior for the Wave 2 native MVP.
- `dashboard_and_entries_mvp.md`: dashboard summary and entries list behavior for the Wave 2 native MVP.

## Scope

- native screen behavior, loading/empty/error states, and iPhone-specific UX notes
- mobile implementation details that do not belong in the repo-wide docs tree
